# 🎰 로또번호 추첨기 (Lotto Generator)

다크모드 및 화이트모드 지원과 실제 로또 공 색상이 반영된 로또 번호 추첨 서비스입니다.

## ✨ 주요 기능
- **로또 번호 생성**: 1~45 무작위 중복 없는 6개 번호 생성 및 오름차순 정렬
- **다크 모드 / 화이트 모드**: 상단 해/달 아이콘 버튼으로 테마 전환 (사용자 설정 저장)
- **로또 공 대표 색상 적용**: 1~10(노랑), 11~20(파랑), 21~30(빨강), 31~40(회색), 41~45(초록)
- **글래스모피즘 & 애니메이션**: 세련된 UI와 번호 생성 시 등장 애니메이션

## 📁 파일 구성
- `index.html`: 서비스 메인 레이아웃 및 테마 토글 버튼
- `style.css`: 테마 변수(CSS Variables), 글래스모피즘 스타일, 로또 공 및 반응형 디자인
- `main.js`: 로또 추첨 로직, 공 색상 분기, 테마 전환 및 `localStorage` 연동
